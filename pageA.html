    Position dan z-index

    Static

    Nilai default elemen


    Relative

    Maju 1 dimensi dan dapat dipindah (left;top;bottom;right) tetapi ruang yang sebelumnya ditempati kosong dan tidak bisa diisi oleh elemen lain.Value ini jarang digunakan


    Absolute

    Maju 1 dimensi dan dapat dipindah (left;bottom;top;right) tetapi runag kosong sebelumnya yang ditempati akan diisi oleh elemen lain


    Fixed

    Membuat elemen dapat dipindah (left;bottom;top;right) tetapi tidak akan bergeser ketika halaman web discroll ke bawah


    Contoh penggunaan

    Value

    Value adalah sebuah angka

    Unit

    Unit adalah satuan

    Jenis-jenis unit

    • Integer bilangan bulat
    • Number pecahan
    • Percentage persen
    • Dimension

    • Angle (deg,rad,grad,turn)
    • Time (s,ms)
    • Resolution (dpi,dpcm,dppx)

    em akan mencari font-size dari elemen bersangkutan atau parentnya sampai menemukan unit absolute lalu dikalikan. Penggunaanya untuk length element

    rem root em akan mencari font-size dari root atau html. Penggunaanya untuk font-size

    Value min(), max(), dan clamp()

    Min()

    Value min akan membandingkan 2 nilai yang lebih kecil
    Jika viewportnya kecil maka browser akan mengambil nilai terkecil
    Jika viewportnya besar maka browser akan mengambil nilai terbesar
    Nilai yang tidak dihitung akan diabaikan oleh browser


    Max

    Value max akan membandingkan 2 nilai yang lebih besar
    Jika viewportnya kecil maka browser akan mengambil nilai terbesar
    Jika viewportnya besar maka browser akan mengambil nilai terkecil
    Nilai yang tidak dihitung akan diabaikan oleh browser

    Clamp

    Value clamp akan membandingkan 3 nilai yaitu min,mid,max
    Value ini akan mencari nilai yang sesuai
